# 問題を出題

既存の問題からランダムに問題を選び、新しいセッションを作成して出題します。

## 実行指示

`templates/question_template.md`の「問題の出題（Agent向け）」セクションに記載されている手順に従って、既存の問題からランダムに問題を選択し、新しいセッションを作成してください。

- `$ARGUMENTS`に引数の値が指定されている場合: その数字分だけ問題を選択します（例: `$ARGUMENTS`が`2`の場合は2問）
- 引数が指定されていない場合: 1問だけ選択します

詳細な手順とファイル形式については、`templates/question_template.md`を参照してください。

## セッション作成後の処理

セッション作成後、以下のファイルをCursor上で開いてください：

- `question.md` - 出題された問題
- `answer.md` - 回答用テンプレート
- `feedback.md` - 添削結果用（初期状態は空）

開く方法：`open -a "Cursor" <セッションディレクトリ>/question.md <セッションディレクトリ>/answer.md <セッションディレクトリ>/feedback.md`
